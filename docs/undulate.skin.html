4.4.3. undulate.skin module

This define the style of the drawing it follows the principle of the css but accept only a subset: font: size, style, variant, weight, stretch, align, family fill: color, opacity stroke: color, width, opacity, css.LineCap, css.LineJoin, mitterlimit, dasharray, opacity

colors should always be in rgba with value from 0—255
